# 특이사항
코다리찜은 해물찜류에 포함시킬 것
시각적으로 분류되게끔 재분류할 것 현재는 이름별로 분류된 느낌이 강함. 예를들어 흰 양념 음식과 빨간 양념이 혼합된 분류면 제대로 분류되지 않을 것. 음식의 형태도 고려되면 좋을듯
